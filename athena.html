Athena Mod Framework

Server-Side Suite for The Isle (Evrima)

The Athena Mod Framework is a comprehensive, server-side modification suite built to solve the biggest challenges facing server owners today: security, management, and customization. By installing Athena, you gain access to powerful tools that run alongside the server, providing control that standard RCON tools cannot match.

Public FAQ

No. The Athena Mod Framework is a server-side license. Players join freely.

Absolutely not. Orion Labs strictly prohibits Pay-to-Win mechanics.

No. Athena is built exclusively for Evrima/Gateway.

Development Roadmap

Our vision for the future of Athena. From advanced AI ecosystems to complete Lua modding support.

Population AI Integration

A completely toggle-able system (requires AI enabled). If species count exceeds a threshold, the server throttles spawn rates for that group, forcing them to hunt players instead of farming AI.

Sandbox Gamemode

Zero delays. Spawn instantly as Adult (or configure to Frail Elder, Elder, Prime Elder). Perfect for events or testing servers.

Custom AI Spawns

Set specific locations and radii for AI spawns. Configure the type (Dino/Animal), growth stage, and even mix multiple types within a single zone.

Enhanced AI Logic

Replacing original controllers with smart, feature-complete logic. AI will act naturally, fight harder, and provide a deeper immersive experience.

Customized Stats

Full control over balance. Edit hunger, stamina, health, weight, thirst, blood, damage, and growth curves to create a unique server meta.

New Mutations

Adding a variety of new mutations ranging from hyper-realistic adaptations to fun, arcade-style buffs to diversify gameplay.

Advanced Plant Spawning

Customize flora logic: keep them in migration zones, spawn by biome, or spawn globally. Enable dynamic spawning based on hunger for small herbs that phases out as they grow.

Custom Migration & Patrol

Fully definable Migration and Patrol zones, allowing you to direct player flow and hotspots exactly where you want them.

Lua Runtime Modding

Long-term goal: Full Lua support. Extend our mod, automate server tasks, and use the Lua wrapper to modify game logic itself!
