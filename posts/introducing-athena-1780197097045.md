# Introducing Athena

Today, we are proud to officially announce Athena, Orion Labs' next generation server management and modification framework for The Isle Evrima.

Athena was created to address some of the biggest challenges server owners face every day. From security concerns and administrative limitations to the need for deeper customization, server operators often find themselves constrained by the tools available to them. Athena was built to change that.

## What is Athena?

Athena is a comprehensive server side modification framework designed to run alongside your server, providing advanced management, security, and customization capabilities.

Unlike traditional RCON based tools, Athena integrates directly with the server environment to provide a greater level of control and flexibility. This allows communities to manage their servers more effectively while creating experiences tailored to their players.

Our goal is simple: provide server owners with the tools they need to build, manage, and protect thriving communities.

## Built for Modern Communities

As communities grow, so do the challenges of running a successful server.

Athena is designed to help server owners:

- Improve server security
- Expand administrative capabilities
- Streamline server management
- Enable deeper customization
- Create unique community experiences
- Reduce reliance on limited external tools

Whether you are operating a small private community or a large public server, Athena provides a foundation for long term growth and stability.

## System Requirements

To ensure a stable and supported installation, Athena currently requires:

### Software and Access

- Windows Server or Windows 10/11 64 bit
- Full desktop and file access to the server
- Latest Evrima update

### Recommended Hardware

- 4 or more CPU cores with strong single thread performance
- 16GB RAM minimum
- 32GB RAM recommended for larger populations
- SSD or NVMe storage
- Stable Gigabit network connection



---



> Athena is designed for dedicated environments where administrators have full control over their server infrastructure.


### Experimental Platforms

While it may be possible to run Athena on Linux through Wine or Proton, or on limited access hosting solutions, these configurations remain untested and unsupported at this time.

For production environments, we strongly recommend a dedicated server, VPS, VDS, or bare metal solution with full administrative access.

## Rollout and Availability

Athena is currently being released in phases as we continue testing, refining features, and expanding support.

To obtain access:

1. Review the Athena EULA.
2. Open a support ticket with our team.
3. Confirm that your hardware meets the required specifications.
4. Once approved, subscribe to the appropriate Athena tier.

This phased rollout allows us to ensure quality, stability, and support as the framework grows.
